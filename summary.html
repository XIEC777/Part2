项目结构汇总

xiec 的 blog — 六个页面的完整架构、功能与技术实现一览

🏗️ 全局架构

共享组件

Header 导航栏

  • 固定定位 (position: fixed, z-index: 99)
  • 毛玻璃效果 (backdrop-filter: blur(12px))
  • 三栏布局：左侧 Logo + 居中导航 + 右侧工具
  • 导航链接下划线 hover 动画
  • 当前页高亮 (is-active)
  • 响应式：移动端简化布局
共享背景

流动色彩 + 毛玻璃

  • 5 个彩色光斑 (.flow-blob)：暖金 / 淡蓝 / 薰衣草 / 天蓝 / 暖粉
  • 独立漂移动画 (flowDrift1-5)，16-22s 循环
  • 全屏毛玻璃覆盖 (blur: 60px)
  • 主页专属：叠加圆点纹理背景
设计系统

视觉风格统一

  • 主色调：#dba460 暖金色 (品牌强调色)
  • 中性色：#111 深黑 / #666-#888 灰 / #aaa 浅灰
  • 字体：PingFang SC, Microsoft YaHei, sans-serif
  • 圆角：12-16px 统一卡片圆角
  • 动画：fadeInUp 入场 + cubic-bezier(.22,1,.36,1) 缓动
  • 页脚：统一布局 © 2026 xiec / 湖南长沙

📄 页面详解

01
index.html

主页

个人博客首页 / 品牌落地页

结构

  • <section class="hero"> 全屏 (100vh) Hero 区域
  • .hero__brand "hello" 金色品牌文字
  • .hero__title 大标题 "xiec 的 blog" (clamp 86-122px)
  • .hero__countdown 学校名 + 地点
  • #cursor 400px 黑色圆形自定义光标

功能与交互

  • 自定义大圆形光标跟随鼠标移动
  • 标题文字 3D 朝向效果 (script.js)
  • 导航栏区域自动隐藏自定义光标
  • 主页专属波点背景 (radial-gradient 圆点纹理)
  • 所有文字元素 fadeInUp 入场动画
  • 光斑持续漂移营造流动氛围

技术要点

  • body.page-home 触发主页专属样式
  • CSS 变量未使用，直接硬编码颜色值
  • 自定义光标通过 JS mousemove 事件驱动
  • 响应式：移动端隐藏自定义光标
fadeInUp 动画自定义光标毛玻璃背景全屏 Hero
02
works.html + works.css + works.js

作品

创意作品集展示 / 分类筛选 / 灯箱预览

结构

  • .works__hero 页面标题区 (PORTFOLIO / My Works)
  • .works__filters 5 个分类筛选按钮 (全部/建模/拍摄/调色/其他)
  • .works__grid 3 列 CSS Grid (响应式: 2列→1列)
  • 20 张 .works__card：8 建模 + 7 拍摄 + 2 调色视频 + 其他占位
  • .lightbox 全屏灯箱模态框 (前后导航/关闭/信息展示)
  • .works__more 加载更多按钮

功能与交互

  • 分类筛选：点击按钮通过 data-category 显示/隐藏卡片
  • 灯箱预览：点击卡片打开全屏大图，支持键盘/按钮前后导航
  • 卡片 hover: 上移 6px + 阴影 + 金色边框 + 图片放大 1.05x
  • 视频作品自动循环静音播放
  • 所有卡片错开延迟的 fadeInUp 入场动画

技术要点

  • 筛选逻辑通过 works.js 实现，使用 .is-hidden 类
  • 灯箱 opacity + visibility 过渡，图片 scale(0.9)→scale(1)
  • 4:3 固定宽高比媒体容器 (aspect-ratio: 4/3)
  • lazy loading 图片优化性能
CSS Grid分类筛选灯箱模态框视频嵌入lazy load
03
social.html + social.css

社交

社交平台链接卡片 / 外部跳转

结构

  • .social__hero 标题区 (SOCIAL / 社交平台)
  • .social__grid 2 列网格 (响应式 1 列)
  • 4 张 .social__card (均为 <a> 标签)
  • 每张卡片：彩色图标 + 标签/标题/描述 + 跳转箭头

功能与交互

  • 平台卡片：GitHub / 小红书 / Bilibili / 抖音
  • 卡片为 <a> 链接，target="_blank" 新窗口跳转
  • hover: 上移 4px + 金色边框 + 渐变背景 + 图标放大 + 标题变色
  • 箭头 hover: 金色变色 + 右上方位移

技术要点

  • 伪元素 ::before 实现 hover 渐变背景
  • 平台图标使用内联 SVG，自定义品牌色背景
  • 简洁的单文件 CSS，无 JS 逻辑
2列网格SVG 图标外部链接渐变背景
04
music.html + music.css

音乐

嵌入外部音乐网站 / iframe 集成

结构

  • iframe.music__frame 嵌入 Hanakos Music
  • 几乎占满视口 (calc(100vh - 80px))
  • 底部页脚 (版权 + 地点)

功能与交互

  • 嵌入 https://forum.hanakos.cc/music
  • 允许全屏 (allow="fullscreen")
  • lazy loading 延迟加载
  • 圆角 12px 白色背景

技术要点

  • 纯 iframe 嵌入，无复杂 JS 逻辑
  • 最简洁的页面结构
  • 移动端自适应高度
iframe 嵌入第三方集成全屏支持
05
zixi.html (独立单文件 ~2000行)

自习

MBTI 沉浸式自习室 / 番茄钟计时 / 数据统计

结构

  • 选择页 (#selection)：MBTI 16 人格类型卡片选择 + 每日目标设置
  • 学习界面 (#studyUI)：顶部栏 (时钟/场景/音量) + 中央计时器 + 底栏
  • 侧边导航 (.side-nav)：6 个页面链接，鼠标悬停滑出
  • 滑出面板：待办 / 日记 / 成就 / 统计 / 设置 (左右滑出)
  • 弹出层：退出报告 / 护眼提醒 / 完成通知 / 成就弹窗
  • 背景系统：Canvas 粒子特效 + 场景图片 + 光斑 + 暗角

功能与交互

  • 16 种 MBTI 人格：每种对应独特场景 (森林/海洋/星空等)
  • 番茄钟计时：正计时 / 倒计时 / 自定义时长
  • BGM 系统：多种场景白噪音 + 音量控制
  • 待办清单：添加/完成/删除任务，进度统计
  • 学习日记：每次完成后记录心得
  • 成就系统：累计番茄数解锁成就徽章
  • 数据统计：总时长/天数/连续天数/时段热力图/每日目标
  • 数据导入导出：JSON 格式备份恢复
  • 护眼提醒：每 20 分钟弹出休息提示
  • 全屏模式：沉浸式专注体验
  • 专注模式：隐藏 UI 仅显示计时器
  • 场景随机切换：一键随机探索

技术要点

  • 单文件架构：~2000 行 HTML/CSS/JS 内联
  • CSS 变量主题系统：--accent / --glass / --border / --t1-t3 / --surface
  • Canvas 粒子动画：背景动态粒子效果
  • localStorage 持久化：所有数据本地存储
  • 昼夜模式：body.light 类切换完整主题
  • 环形进度条：SVG circle 实现计时可视化
  • MBTI 分组：分析师/外交家/守护者/探险家四组
  • 热力图：展示每日学习时段分布
番茄钟MBTICanvas 粒子localStorageCSS 变量成就系统数据统计热力图昼夜主题全屏模式
