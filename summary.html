01
index.html + styles.css + script.js

主页

个人博客首页 / 品牌落地页 / 自定义光标与字符 3D 旋转

结构

  • <section class="hero"> 全屏 (100vh) Hero 区域
  • .hero__brand "hello" 金色品牌文字 (fadeInUp 入场)
  • .hero__title 大标题 "xiec 的 blog" (clamp 86-122px)
  • .hero__countdown 学校名 + 地点
  • #cursor 400px 黑色圆形自定义光标
  • .hero__char 每个标题字符独立 span，支持 3D 变换

功能与交互

  • 自定义大圆形光标：400px 黑色圆跟随鼠标移动，平滑过渡
  • 标题文字 3D 朝向：每个字符随光标位置独立旋转 (rotateX/Y)
  • 标题切换：光标移入标题区域切换为「你好 我是谢承轩」
  • 导航栏区域自动隐藏自定义光标 (pointer-events 恢复默认)
  • 主页专属波点背景：radial-gradient 圆点纹理叠加毛玻璃
  • 所有文字元素 fadeInUp 错位入场动画
  • 5 个彩色光斑持续漂移营造流动氛围

技术要点

  • body.page-home 类触发主页专属样式（波点背景、自定义光标）
  • 自定义光标通过 JS mousemove 事件驱动 (transform translate)
  • 字符 3D 通过 perspective + rotateX/Y + transform-origin
  • 标题切换使用 opacity + pointer-events 过渡
  • 响应式：移动端隐藏自定义光标 (display: none)
fadeInUp 动画自定义光标字符 3D 旋转毛玻璃背景全屏 Hero标题切换视差效果
02
works.html + works.css + works.js

作品

创意作品集展示 / 3D 轮播 / 分类筛选 / 灯箱预览 / 鼠标视差

结构

  • .works__hero 页面标题区 (PORTFOLIO / My Works)
  • .works__carousel 3D 堆叠轮播图 (7 张精选作品)
  • .works__filters 5 个分类筛选按钮 (全部/建模/拍摄/调色/其他)
  • .works__grid 3 列 CSS Grid (响应式: 2列→1列)
  • 20 张 .works__card：8 建模 + 7 拍摄 + 2 调色视频 + 其他占位
  • .lightbox 全屏灯箱模态框 (前后导航/关闭/信息展示)
  • .works__more 加载更多按钮

功能与交互

  • 3D 堆叠轮播：当前卡片居中 + 左右两侧卡片缩放倾斜
  • 轮播卡片旋转: rotateY(±15°) + rotateZ(±3°) 3D 倾斜
  • 毛玻璃卡片：backdrop-filter blur + 半透明边框 + 内阴影
  • 鼠标视差：光标移动时卡片 3D 旋转跟随 (倾斜 ±20°)
  • 当前卡片动态阴影随鼠标方向偏移
  • 轮播点击灯箱：任意轮播卡片均可点击打开全屏预览
  • 自动播放 (4s)、hover 暂停、触摸滑动、键盘导航 (←→)
  • 分类筛选：点击按钮通过 data-category 显示/隐藏卡片
  • 灯箱预览：点击网格卡片打开全屏大图，支持键盘/按钮前后导航
  • 视频作品自动循环静音播放
  • 所有卡片错开延迟的 fadeInUp 入场动画

技术要点

  • 轮播使用 class 状态机 (is-active/is-prev/is-next/is-hidden)
  • 3D 透视: perspective(1200px) + transform-origin
  • 内层 .carousel__inner 独立控制圆角与 overflow
  • 视差通过 mousemove 计算相对位置驱动 transform
  • 轮播与网格卡片通过图片 src 匹配联动灯箱
  • 筛选逻辑使用 .is-hidden 类控制显隐
  • 灯箱 opacity + visibility 过渡，图片 scale 动画
  • lazy loading 图片优化性能
3D 轮播鼠标视差CSS Grid分类筛选灯箱模态框毛玻璃视频嵌入lazy load
03
social.html + social.css + social.js

社交

社交平台链接卡片 / 扑克牌摊开动画 / hover 抽出效果

结构

  • .social__hero 标题区 (SOCIAL / 社交平台)
  • .social__grid 扑克牌扇形层叠布局 (flex + fixed 定位)
  • 8 张 .social__card (均为 <a> 标签)
  • 每张卡片：彩色图标 + 标签/标题/描述 + 跳转箭头

功能与交互

  • 平台卡片：GitHub / 小红书 / Bilibili / 抖音 / QQ / 微信 / 微博 / 网易云音乐
  • 扑克牌摊开动画：卡片从中心堆叠 → 缩放 → 扇形展开 (rAF 驱动)
  • hover 抽出效果：当前卡片向上移出牌堆 (~140px) + 轻微放大
  • 左右邻居卡片自动上移并向外推开，为当前卡片腾出空间
  • 卡片为 <a> 链接，target="_blank" 新窗口跳转
  • hover: 金色边框 + 渐变背景 + 图标放大 + 标题变色 + 深阴影
  • 箭头 hover: 金色变色 + 右上方位移

技术要点

  • 入场动画：单一 rAF 循环驱动 4 阶段动画 (渐入→放大→保持→摊开)
  • 动画阶段通过 transform/opacity 实现，避免 layout thrashing
  • 卡片使用 fixed 定位 + transform translate3d 实现扇形排列
  • 动画结束后切换到 left/top 定位 + CSS transition 接管 hover
  • resize 响应重新计算卡片位置
  • 伪元素 ::before 实现 hover 渐变背景
  • 平台图标使用内联 SVG，自定义品牌色背景
扑克牌动画rAF 驱动扇形摊开hover 抽出SVG 图标外部链接fixed 定位3D transform

05
zixi.html (独立单文件 ~2000行)

MBTI 沉浸自习室

MBTI 沉浸式自习室 / 番茄钟计时 / 数据统计

结构

  • 选择页 (#selection)：MBTI 16 人格类型卡片选择 + 每日目标设置
  • 学习界面 (#studyUI)：顶部栏 (时钟/场景/音量) + 中央计时器 + 底栏
  • 侧边导航 (.side-nav)：6 个页面链接，鼠标悬停滑出
  • 滑出面板：待办 / 日记 / 成就 / 统计 / 设置 (左右滑出)
  • 弹出层：退出报告 / 护眼提醒 / 完成通知 / 成就弹窗
  • 背景系统：Canvas 粒子特效 + 场景图片 + 光斑 + 暗角

功能与交互

  • 16 种 MBTI 人格：每种对应独特场景 (森林/海洋/星空等)
  • 番茄钟计时：正计时 / 倒计时 / 自定义时长
  • BGM 系统：多种场景白噪音 + 音量控制
  • 待办清单：添加/完成/删除任务，进度统计
  • 学习日记：每次完成后记录心得
  • 成就系统：累计番茄数解锁成就徽章
  • 数据统计：总时长/天数/连续天数/时段热力图/每日目标
  • 数据导入导出：JSON 格式备份恢复
  • 护眼提醒：每 20 分钟弹出休息提示
  • 全屏模式：沉浸式专注体验
  • 专注模式：隐藏 UI 仅显示计时器
  • 场景随机切换：一键随机探索

技术要点

  • 单文件架构：~2000 行 HTML/CSS/JS 内联
  • CSS 变量主题系统：--accent / --glass / --border / --t1-t3 / --surface
  • Canvas 粒子动画：背景动态粒子效果
  • localStorage 持久化：所有数据本地存储
  • 昼夜模式：body.light 类切换完整主题
  • 环形进度条：SVG circle 实现计时可视化
  • MBTI 分组：分析师/外交家/守护者/探险家四组
  • 热力图：展示每日学习时段分布
番茄钟MBTICanvas 粒子localStorageCSS 变量成就系统数据统计热力图昼夜主题全屏模式
06
suggest.html + suggest.css

建议

用户反馈表单 / Tally 集成 / 物理卡片拖拽系统

结构

  • .suggest__hero 标题区 (FEEDBACK / 建议反馈)
  • .suggest__form 白色卡片容器 (max-width 680px)
  • Tally 表单 iframe 嵌入
  • 4 张背景预览卡片：主页/作品/社交/音乐 页面实时预览
  • 左下角控制面板：一键归位 + 自动归位开关

功能与交互

  • 嵌入 Tally.so 表单 (rjgllM)
  • 动态加载 Tally 嵌入脚本
  • 2D 物理引擎：重力 + 空气阻力 + 地面摩擦 + 弹性反弹
  • SAT 碰撞检测：分离轴定理检测卡片间碰撞
  • 拖拽甩出：1:1 跟手拖拽 + 松手速度增强 + 旋转拖拽
  • 角速度物理：碰撞产生扭矩 + 地面回正 + 角速度衰减
  • 动态阴影：随卡片高度实时变化
  • 自动归位：静止后自动回到原始 CSS 位置
  • 漂浮动画：未拖拽时卡片上下轻微浮动

技术要点

  • SAT 分离轴定理：4 顶点 OBB 碰撞检测 + 法线分离
  • 碰撞冷却：50ms 冷却防止同帧反复抖动
  • 碰撞分离余量：60px 额外分离确保不再重叠
  • 统一物理循环：全局 rAF 驱动所有物理卡片
  • GPU 优化：物理模拟时禁用 backdrop-filter
  • 触摸支持：完整的 touchstart/move/end 事件处理
  • 第三方表单集成：异步脚本加载 + 回调初始化
Tally 表单物理引擎SAT 碰撞拖拽甩出旋转拖拽动态阴影自动归位iframe 预览
